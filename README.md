# Projeto de Visão Computacional com IA

Projeto desenvolvido utilizando TensorFlow e redes neurais convolucionais (CNN) para classificação automática de imagens de gatos e cachorros.

## Tecnologias Utilizadas
- Python
- TensorFlow
- OpenCV
- NumPy
- Matplotlib

## Funcionalidades
- Treinamento de rede neural
- Classificação de imagens
- Processamento de datasets
- Inferência com IA

## Objetivo
Projeto criado para estudos de Inteligência Artificial e Visão Computacional.
